How to Use the Tracker

Graph

Scroll Zoom in / out
Drag background Pan the graph
Drag a node Reposition a region node
Hover a connection Show the loading zone names on each end

Color Mode

Toggle top-left checkbox Color-codes region nodes by accessibility
Green node Has at least one reachable, incomplete item
Red node All items are currently blocked
Gray node All items are done
Amber outline Your current region

Sidebar

◀ / ▶ tab (right edge) Show or hide the sidebar panel
Left-click an item Expand / collapse the access rule explanation
Right-click any item Opens a small menu with context actions.
Toggle Done (locations) Grays out the location the same as a server-confirmed pickup. Works in both directions — also un-grays AP-confirmed locations.
Suggest Rule Change Opens a text box showing the current rule. Type what you think the rule should be. All suggestions are saved to a .txt file — close the tracker by pressing Ctrl+C in the console window to generate it.

Search

Type in search box Find any location or zone by name
Select a result Highlights the shortest path from your current region (or Rogueport Center as fallback) on the graph in purple, with the target node in amber
↑ ↓ in search Navigate the suggestion dropdown
Enter Select the highlighted suggestion
Esc Clear search and remove path highlight
